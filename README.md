# API de Clientes - Spring Boot

## 📌 Sobre o projeto
API REST desenvolvida para gerenciamento de clientes, com operações CRUD completas.

Este projeto foi criado com foco em praticar desenvolvimento backend com Java e Spring Boot.

## 📌 Tecnologias utilizadas
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## 📌 Funcionalidades
- ✅ Criar cliente
- ✅ Listar clientes
- ✅ Atualizar cliente
- ✅ Deletar cliente

## 📌 Endpoints

### 🔹 Listar clientes
GET /clientes

### 🔹 Criar cliente
POST /clientes

### 🔹 Atualizar cliente
PUT /clientes/{id}

### 🔹 Deletar cliente
DELETE /clientes/{id}
