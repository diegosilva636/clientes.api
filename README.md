# API de Clientes - Spring Boot

🚀 Projeto desenvolvido para gerenciamento de clientes utilizando Java e Spring Boot.

## 📌 Tecnologias utilizadas
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## 📌 Funcionalidades
- ✅ Criar cliente
- ✅ Listar clientes
- ✅ Atualizar cliente
- ✅ Deletar cliente

## 📌 Endpoints

### 🔹 Listar clientes
GET /clientes

### 🔹 Criar cliente
POST /clientes

### 🔹 Atualizar cliente
PUT /clientes/{id}

### 🔹 Deletar cliente
DELETE /clientes/{id}
